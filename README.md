## Steps 

Clone the CPSC-349-Project-4 repository locally:

```shell-session
git clone https://github.com/cardogg/CPSC-349-Project-4.git
```

Install and start Project 4:

```shell-session
cd CPSC-349-Project-4/
npm install
npm start
```

Open the [Site][1]

Open the [PocketBase Admin UI][2]
